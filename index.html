Contribute via Pull Request

This site is static and community-driven. Add your prompt, media file, and metadata in a PR — once merged, it appears in the gallery automatically.

  1. Fork the repository and create a branch.
  2. Add your image or video to assets/images/ or assets/videos/.
  3. Copy submissions/template.json and add your entry to data/prompts.json.
  4. Open a pull request. CI will validate your submission.
