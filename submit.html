Contribute

Submit a prompt

Share your AI-generated image or video along with the exact prompt you used. Submissions are added through a pull request and reviewed before going live.

How it works

  1. 1

    Fork & branch

    Create a branch in your fork of the repository.

  2. 2

    Upload your media

    Add your file to assets/images/ or assets/videos/.

  3. 3

    Add your entry

    Append your JSON entry to data/prompts.json using the builder below.

  4. 4

    Open a pull request

    CI validates your submission automatically. Once merged, it appears in the gallery.

Accepted formats

  • Images: jpg, png, webp, gif, svg
  • Videos: mp4, webm, mov

Before you submit

  • Use the exact prompt that produced the output
  • Pick a unique, descriptive ID
  • Run node scripts/validate.js locally
